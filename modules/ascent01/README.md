# Module 1: Foundations — Statistics, Probability & Data Fluency

**Kailash**: kailash-ml (DataExplorer, PreprocessingPipeline, ModelVisualizer) | **Scaffolding**: 70%

## Lecture (3h)

- **1A** Statistical Foundations: exponential family, Bayesian estimation, MLE, hypothesis testing (power analysis, FDR), bootstrapping (BCa)
- **1B** Polars: Arrow backend, lazy evaluation, expression API, window functions
- **1C** EDA at Scale: DataExplorer (8 alert types, async profiling), PreprocessingPipeline, ModelVisualizer

## Lab (3h) — 8 Exercises

1. First data exploration: Polars basics on HDB resale data
2. Filtering and transforming HDB transactions
3. Functions, aggregation, and group-by analysis
4. Joins and multi-table data enrichment (MRT, schools)
5. Window functions and price trends over time
6. Interactive data visualization with ModelVisualizer
7. Automated data profiling with DataExplorer
8. Capstone: data cleaning and end-to-end EDA pipeline

## Datasets

Singapore HDB Resale (15M+), Economic Indicators (CPI/employment/FX), Taxi Trips (LTA), E-commerce A/B Test
